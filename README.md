# Java Developer Companion (`demo-java-companion`)

An enterprise-grade GitHub Copilot extension designed to streamline Java development workflows. This plugin provides a dedicated AI agent to generate pristine unit tests and a specialized skill to reverse-engineer and document complex Java source code.

---

## 🚀 Features

*   **`@demo-java-test-creator` (Agent):** A dedicated chat persona that writes robust, architecture-compliant JUnit 5 and Mockito unit tests following the AAA (Arrange, Act, Assert) pattern.
*   **`/demo-java-describer` (Skill):** A dynamic code-analysis skill that breaks down legacy or complex Java components into high-level structural overviews, design patterns, and control flows.
